This program requires opening a FAT formatted image in order to work.

Folder structure

Documentation
  - Contains report

src
  - Contains code and README (this file) of the program
  - contains the classes, makefile in order to run it all
  - compile.sh is a bash script that will compile and run fat32.img
  

Program Use
  - To run the program after using make do ./fmod <img name>
    - example usage: ./fmod fat32.img

Notes
  - This program skips long file names as per instructor guidelines
